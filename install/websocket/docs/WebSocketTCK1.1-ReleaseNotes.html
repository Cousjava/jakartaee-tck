JavaTM API for WebSocket Technology Compatibility Kit, Version 1.1
Release Notes, August 2014

Kit Contents

The JavaTM API for WebSocket, Version 1.1 Technology Compatibility Kit (TCK) includes the following items:

  • JavaTM API for WebSocket TCK tests signature, API, and End-to-End tests:

    • A signature test that checks that all of the public APIs are supported in the JavaTM API for WebSocket Version 1.1 implementation that is being tested

    • API tests for the public APIs under the javax.websocket and javax.websocket.server packages

    • Specification tests for testable assertions required by the WebSocket 1.1 Specification

WebSocket TCK Facts

The test suite bundle contains the following:

  • 664 total tests (663 tests + 1 signature test)

Platform Notes

The JavaTM API for WebSocket TCK tests have been built and tested with JavaTM Platform, Standard Edition 7 (Java SE 7).

The JavaTM API for WebSocket TCK tests have been run on the following platforms:

  • Solaris 11 Express (x64 architecture)
  • MAC OS X Mountain Lion (10.8.1+)
  • Windows XP, Windows 7, Windows 8
  • Oracle Linux 6
  • RedHat Enterprise Linux 6
  • Ubuntu Linux 12.10
  • Suse Enterprise Linux 11

The JavaTM API for WebSocket TCK tests have been run against the following JavaTM API for WebSocket 1.1 Reference Implementations:

  • Oracle JavaTM API for WebSocket Reference Implementation Version 1.1

Installing, Setting Up, and Running the JavaTM API for WebSocket TCK

Refer to the JavaTM API for WebSocket TCK 1.1 User's Guide for complete instructions on installing, setting up, and running the JavaTM API for WebSocket TCK. The online version of the JT Harness version 5.0 documentation is available here.
