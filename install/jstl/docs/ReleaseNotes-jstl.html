A Standard Tag Library for JavaServer PagesTM (JSTL) Technology Compatibility Kit, Version 1.2
Release Notes, March 2010

Kit Contents

A Standard Tag Library for JavaServer Pages(JSTL), Version 1.2, Technology Compatibility Kit (TCK) includes the following items:

  • JSTL TCK signature, API and specification tests:

    • A signature test that checks that all of the public APIs are supported or defined in the Standard Tag Library for JavaServer Pages TM Version 1.2 implementation that is under test.

    • Specification tests for the testable assertions required by the JSTL Revision 1.2 Specification.

    • API tests for the public APIs and language elements defined by the JSTL 1.2 Specification.

Platform Notes

The JSTL 1.2 TCK tests and the JSTL 1.2 Reference Implementation (RI) can be run with either JavaTM Platform, Standard Edition (Java SE) version 1.5 or JDK 6, depending on the web container in which you run the tests. See Reference Runtime for JSTL 1.2 for additional information.

Refer to the A Standard Tag Library for JavaServer PagesTM User's Guide for platform compatibility requirements and notes.

Reference Runtime for A Standard Tag LibraryTM 1.2

The designated Reference Runtime for compatibility testing of implementations based upon the Standard Tag Library for JavaServer PagesTM Specification 1.2 is as follows:

  • J2SE 1.5 or JDK 6, referred to in the Standard Tag Library for JavaServer PagesTM 1.2 documentation as the appropriate <JDK_Version>

    Note - The J2SE 1.5 software can only be used with a non-Java EE 6 profile/application server (for example, Tomcat 6.x, which supports Servlet 2.5); JDK 6 must be used with a Java EE 6 profile/application server, such as the GlassFish v3 server.

  • Sun Reference Implementation (RI) of A Standard Tag Library for JavaServer PagesTM 1.2 available as a standalone implementation from Java Partner Engineering or included as part of the Java EE 6 RI or the Java EE 6 Web Profile RI, neither of which have been finalized at the time of this writing but are also available from Java Partner Engineering

    Note - Java EE 6 is not required; this TCK only requires a container that supports A Standard Tag Library for JavaServer PagesTM 1.2 and its requirements.

  • A Standard Tag Library for JavaServer PagesTM TCK tests run on the following platforms:

    • Solaris 9 Operating System on Sparc and x86
    • Solaris 10 Operating System on Sparc and Opteron
    • Windows XP Professional Edition
    • Windows Vista
    • RedHat Linux 9.0
    • RedHat Linux AS 3.0 (including Update 4)
    • MacOS X 10.5.7g
