JSON-B Technology Compatibility Kit, Version 1.0
Release Notes, May 2017

Note

This is a patch release of the TCK. The Release Notes file that is posted on the Java Partner Engineering Web site contains the most current information about the TCK, including any last minute changes. Always check that file for the most up-to-date information.

Also note that this patch is a full, comprehensive release of the TCK bundle. You do not need to install any previous TCK bundles before installing this patch bundle.

Kit Contents

The JSON-B, Version 1.0 Technology Compatibility Kit (TCK) includes the following items:

  • JSON-B TCK tests signature, API, and end-to-end tests:

    • A signature test that checks that all of the public APIs are supported in the JSON-B Version 1.0 implementation that is being tested

    • API tests for all of the public APIs under the javax.xml.soap package

    • Specification tests for the testable assertions required by the JSON-B 1.0 Specification.

JSON-B 1.0 TCK Facts

The test suite bundle contains the following:

  • 271 total tests

Platform Notes

The JSON-B TCK tests have been built and tested running on the JavaTM Standard Edition 8 (Java SE 8) platform.

The JSON-B TCK tests run on the following platforms:

  • Oracle Linux 7.1
  • Windows 10

Installing, Setting Up, and Running the JSON-B TCK

Refer to the JSON-B TCK 1.0 User's Guide for complete instructions on installing, setting up, and running the JSON-B TCK. The online version of the JT Harness version 5.0 documentation is available here.
