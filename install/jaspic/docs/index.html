JASPIC TCK 1.1 Documentation

The JavaTM Authentication SPI for Containers (JASPIC) Technology Compatibility Kit (TCK) 1.1 documentation includes the following:

  • The JASPIC TCK 1.1 Release Notes provide late-breaking information about the JASPIC TCK, Version 1.1.

  • The JASPIC 1.1 Technology Compatibility Kit User's Guide provides the information you need to install, configure, and run the JASPIC TCK. The User's Guide also contains the rules you need to pass for certification. The user guide is available on the Java Partner Engineering Web site

  • The API documentation for the porting interfaces provides information about the porting packages you need to implement for this TCK.

  • The Javadoc Assertion List lists all the javadoc assertions from JASPIC 1.1 Specification.

  • The Specification Assertion List lists all the specification assertions from JASPIC 1.1 Specification.
