JMS Technology Compatibility Kit, Version 2.0
Release Notes, May 2013

Kit Contents

The JMS, Version 2.0 Technology Compatibility Kit (TCK) includes the following items:

  • JMS TCK tests signature, API, and End-to-End tests:

    • A signature test that checks that all of the public APIs are supported in the JMS Version 2.0 implementation that is being tested

    • API tests for all of the public APIs under the javax.jms package

    • End-To-End tests that demonstrate sending/receiving JMS messages to/from a QUEUE or TOPIC

JMS TCK Facts

The test suite bundle contains the following:

  • 918 total tests

Platform Notes

The JMS TCK tests have been built and tested with JavaTM Platform, Standard Edition 7 (Java SE 7).

The JMS TCK tests have been run on the following platforms:

  • Solaris 11 Express (x64 architecture)
  • MAC OS X Mountain Lion (10.8.1+)
  • Windows XP, Windows 7, Windows 8
  • Oracle Linux 6
  • RedHat Enterprise Linux 6
  • Ubuntu Linux 12.10
  • Suse Enterprise Linux 11

The JMS TCK tests have been run against the following JMS 2.0 Reference Implementations:

  • Oracle JMS Reference Implementation Version 2.0
  • Oracle Open Message Queue 5.0

Refer to the JMS TCK 2.0 User's Guide for complete instructions on installing, setting up, and running the JMS TCK. The online version of the JT Harness version 5.0 documentation is available here.
