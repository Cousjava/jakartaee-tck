Java EE Security API Technology Compatibility Kit, Version 1.0
Release Notes, September 2017

Kit Contents

The Java EE Security API, Version 1.0, Technology Compatibility Kit (TCK) includes the following items:

  • Java EE Security API TCK documentation:

    • These Release Notes can be found in both the TCK bundle and the documentation bundle.

  • Java EE Security API TCK Signature, API, and specification tests:

    • A signature test that checks that all of the public APIs are supported or defined as specified in the Java EE Security API Version 1.0 implementation under test.

    • API tests for the public APIs and language elements defined by the Java EE Security API 1.0 Specification.

    • Specification tests for the testable assertions required by the Java EE Security API 1.0 Specification.

Platform Notes

The Java EE Security API TCK tests have been built and tested with the Oracle GlassFish Enterprise Server v5 running on the JavaTM Platform, Standard Edition 8 (Java SE 8) platform. If you are using a different Web server, you will need to make adjustments to the procedures outlined in this document.

The Java EE Security API tests run on the following platforms:

  • Oracle Linux 7.1
  • Microsoft Windows 10

Installing, Setting Up, and Running the Java EE Security API TCK

Refer to the Java EE Security API 1.0 Technology Compatibility Kit User's Guide for complete instructions on installing, setting up, and running the Java EE Security API TCK. The online version of the JT Harness version 5.0 documentation is available here.
