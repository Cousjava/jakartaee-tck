Common Annotations for Java (CAJ) Technology Compatibility Kit, Version 1.3
Release Notes, August 2016

Kit Contents

The Common Annotations for Java (CAJ), Version 1.3 Technology Compatibility Kit (TCK) includes the following items:

  • CAJ TCK Signature Tests: Check that all public APIs are supported and/or defined as specified in the Common Annotations for Java Version 1.3 implementation under test.

Platform Notes

The CAJ TCK tests have been built and tested with JavaTM Platform, Standard Edition 8 (Java SE 8).

The CAJ TCK tests have been run on the following platforms:

  • MAC OS X El Capitan (10.9.5+)
  • Oracle Linux 7

The CAJ TCK tests have been run against the following Common Annotations for Java 1.3 Reference Implementations:

  • Oracle Common Annotations for Java Reference Implementation Version 1.3

Installing, Setting Up, and Running the CAJ TCK

Refer to the Common Annotations for Java TCK 1.3 User's Guide for complete instructions on installing, setting up, and running the CAJ TCK. The online version of the JT Harness version 5.0 documentation is available here.
