JASPIC TCK 1.0 Documentation

The Java Authentication SPI for Containers (JASPIC) Technology Compatibility Kit (TCK) 1.0 documentation includes the following:

  • The JASPIC TCK 1.0 Release Notes provide late-breaking information about the JASPIC TCK, Version 1.0.

  • The JASPIC 1.0 Technology Compatibility Kit User's Guide provides the information you need to install, configure, and run the JASPIC TCK.

  • The API documentation for the porting interfaces provides information about the porting packages you need to implement for this TCK.

  • The Javadoc Assertion List lists all the javadoc assertions from JASPIC 1.0 Specification.

  • The Specification Assertion List lists all the specification assertions from JASPIC 1.0 Specification.
